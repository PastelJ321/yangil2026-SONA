현재 구조에서 GitHub에 올리는 방법

메인 페이지는 저장소 맨 위의 index.html이고, 학생 홈페이지는 각자 폴더 안에 있습니다. 그래서 메인 페이지에서는 실제 폴더 경로 그대로 링크해야 합니다.

  • 저장소 루트에 있는 index.html을 GitHub Pages의 시작 페이지로 사용합니다.
  • 학생 페이지는 예를 들어 10609박수현/index.html 같은 실제 경로로 연결합니다.
  • 새 학생 파일을 추가할 때는 members.js에 이름, 설명, 경로만 추가하면 됩니다.
  • GitHub Pages는 경로의 대소문자를 구분하므로 파일명과 확장자를 실제와 똑같이 써야 합니다.
